Word It

This is a computer application designed and programmed in four weeks at XO APPLab Summer Camp. I was the lead developer for a Python program that runs on the One Laptop Per Child (OLPC) laptops. We came up with the idea of a vocabulary game, Word It. The game was completely coded in Python. The game has two main game modes. The first one simply displays the word and definition on the screen for users to study. The second mode tests users' vocabulary by displaying a word, and displaying 4 random definitions with one of them being the correct one. The game also has internet functionality. It searches our server for words. If there is no internet connectivity, it uses the words defined in a separate file in the game's folder.

Color Tap

I developed this app as a student in Youth APPLab. Every student had to develop an app at the end of Youth APPLab's program. This was my submission. Color Tap is a hand-eye coordination game. Circles rapidly change colors and bounce around the screen. The objective is to touch a circle when it is the target color, which also changes. You have a limited amount of time to do so.
